# Alper's Portfolio

A premium portfolio website built with Vite, React, and Framer Motion.

## Features
- **Scroll Snap Navigation**: Smooth full-page scrolling.
- **Interactive Experience Graph**: A radial visualization of professional experience.
- **Premium Aesthetics**: Dark theme with glassmorphism and neon accents.
- **Responsive Design**: Works on Desktop and Mobile.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the local URL (usually http://localhost:5173).

## Customization
- Update `src/components/Home.jsx` to change your bio.
- Update `src/components/ExperienceCircle.jsx` to add more experiences.
- Replace `src/assets/profile.png` with your real photo.
